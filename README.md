# Shortlink DNS Filter

> 대량의 숏링크 주소 차단 필터
---
## 사용법

사용 중인 광고 차단기에 아래 URL로 Shortlink Shield Filter를 추가할 수 있습니다.

> [!NOTE]  
> AdGuard(애드가드), uBlock Origin(유블록 오리진)을 권장합니다.

> [!IMPORTANT]  
> AdGuard(애드가드) 사용자의 경우 DNS 필터로 등록하시면 됩니다.

**Shortlink DNS Filter**:  
```
https://cdn.jsdelivr.net/gh/Null772/Shortlink-DNS-Filter@latest/Shortlink_Filter.txt
```
